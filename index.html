GIFアニメ作成
X / Threads 投稿用ツール

ChatGPT などのAIで生成したコマ割り画像を取り込み、
スマホだけでアニメーションGIFを作れるオフライン対応PWAです。

アプリを開く 導入手順を見る

タップで「GIFアニメ作成」をインストール（アイコンがホーム画面に追加されます）

このアプリでできること

  • 🪄 プロンプト生成
    ChatGPTに貼るだけのコマ割り画像生成用プロンプトを自動で組み立てます。
  • 📋 簡単入力
    クリップボード貼り付け / ファイル選択 / ドラッグ＆ドロップに対応。
  • ✂️ コマ割り編集
    行×列の指定でグリッド分割。境界線の余白調整・並べ替え・除外もできます。
  • 🎞️ GIF生成
    速度・ループ・サイズなど詳細設定可能。逆再生・ピンポンも対応。
  • 👀 プレビュー
    作ったGIFを即その場で再生して確認できます。
  • 💾 端末に保存
    iOSは「写真」へ、Androidは直接ダウンロード。共有シートから各SNSへも投稿可。
  • 💬 テキスト乗せ
    各コマにセリフや字幕を入れられます。
  • 📱 X/Threads最適化
    サイズ・秒数・ループをワンボタンでSNS最適化。
  • 📦 オフライン動作
    一度開けば、機内モードでも全機能が使えます。

スマホへのインストール手順

  1. Safari でこのページを開いてください（Chromeでは追加できません）。
  2. 画面下中央の「共有」ボタン（□に↑のアイコン）をタップ。
  3. メニューを下にスクロールし「ホーム画面に追加」をタップ。
  4. 名前を確認して右上の「追加」をタップ。
  5. ホーム画面のアイコンから起動できます。初回起動後はオフラインでも動作します。

※ Safariのプライベートブラウズ中は追加できません。通常モードでお試しください。

  1. Chrome でこのページを開いてください。
  2. 下部または右上に表示される「ホーム画面に追加」バナー、または「⋮ メニュー」→「ホーム画面に追加」をタップ。
  3. 「追加」をタップで完了。アプリ一覧／ホーム画面にアイコンが作成されます。
  4. 初回起動後はオフラインでも全機能利用可能です。

※ 上部の「ホーム画面に追加」ボタンが表示される場合はそちらが最短です。

使い方の流れ

  1. 1. プロンプトを作る

    アプリの「① プロンプト生成」タブで、コマ数・グリッド形状・キャラクターや動作の説明を入力。生成されたプロンプトをコピーします。

  2. 2. ChatGPT等で画像を生成

    コピーしたプロンプトを ChatGPT / Gemini / Copilot などに貼り付け、コマ割り画像を生成。画像をスマホに保存またはコピーします。

  3. 3. 画像をアプリに取り込む

    「② 画像取込・コマ割り」タブで 貼り付け または ファイル選択 。行数×列数を指定して各コマに分割。並べ替え・除外も可能。

  4. 4. GIFを生成・確認・保存

    「③ 生成・プレビュー・保存」タブで再生時間やループを設定し生成。プレビュー確認後、共有ボタンで端末へ保存します。

X / Threads 投稿時のコツ

  • サイズは512×512前後。XのGIFは最大15MB・ループ再生のため、コマ数とパレット色数を抑えるとスムーズに表示されます。
  • 1〜3秒の短いループがもっとも目に留まります。アプリの「ピンポン再生」で少ないコマでも自然な動きに。
  • 余白を控えめに。タイムライン上では小さく表示されるため、被写体は中央寄せ大きめが◎。
  • テキスト乗せで内容を1〜2語で示すと反応率が上がります。アプリ内のオーバーレイ機能を活用してください。

よくある質問

本当にオフラインで動きますか？

はい。最初にこのページとアプリを一度開くと、必要なファイルがすべて端末にキャッシュされます。以降は機内モードでもGIF作成までフル機能でご利用いただけます。

画像はサーバーに送られますか？

送られません。画像処理とGIF生成はすべて端末内（ブラウザ内）で行われます。ネット接続は不要です。

iOS では写真にどう保存されますか？

「共有」ボタンから iOS の共有シートが開き、「画像を保存」を選ぶと写真アプリに保存されます。動作しない端末向けに、長押し保存もご利用いただけます。

背景透過は使えますか？

白などの単色背景を簡易的に透過化する機能を搭載しています。詳細設定タブから利用できます。
